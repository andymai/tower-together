# Facility Family Docs

These docs define family-specific behavior that sits on top of the shared rules in `FACILITIES.md`, `PEOPLE.md`, `ROUTING.md`, and `ECONOMY.md`.

## Files

- `HOTEL.md`: families `3`, `4`, `5`
- `OFFICE.md`: family `7`
- `CONDO.md`: family `9`
- `COMMERCIAL.md`: families `6`, `0x0c`, `10`
- `ENTERTAINMENT.md`: families `0x12`, `0x1d`
- `LOBBY.md`: family `0x18`
- `PARKING.md`: families `0x0b`, `0x2c`
- `EVALUATION.md`: evaluation visitors and Tower advancement
- `HELPERS.md`: support/helper families and hotel-guest behavior
